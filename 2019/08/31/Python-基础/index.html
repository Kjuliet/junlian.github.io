1. 简介

python 是一个高层次的结合了解释性、面向对象的脚本语言：

  • 解释型语言：意味着开发过程中没有编译这个环节，类似于PHP
  • 面向对象语言：支持面向对象的风格，或代码封装在对象里编程技术。

评估一种语言，需要看它以下几个方面：

  • 是否易于维护；
  • 有没有一个广泛的标准库，这决定了这个语言能否跨平台，在各个系统上的兼容性怎么样；
  • 是否可移植，也就是到了其他平台上依然让它工作；
  • 数据库的接口，是否提供了主要的商业数据库的接口；
  • 可嵌入性，能否把它嵌入到其他语言的程序中，也就是让程序的用户获得“脚本化”的能力。

1.1 python 中文编码

python2 默认的编码格式是ASCII格式，在没修改编码格式时是没有办法正常打印汉字的，所以需要在文件开头加入# -*- coding: UTF-8 -*- 或者 # coding=utf-8，注意等号两边不能有空格。

python3 默认使用utf-8编码，所以不用再指定。如果用编辑器，需要设置py文件的存储格式为UTF-8。

1.2 交互式编程

交互式编程是指不需要创建脚本文件，通过python解释器的交互模式来编写代码。比如在linux命令行输入Python就可以启动交互式编程。

1.3 脚本式编程

通过脚本参数来调用解释器执行脚本，当脚本执行完成，解释器失效。

2. 基本语法

2.1 Python 标识符

  • 标识符由字母、数字、下划线组成，但不能以数字开头；
  • 区分大小写；
  • 以下划线开头的标识符是有特殊含义的：
    • 以单下划线开头_foo代表不能直接访问的类属性，需要通过类提供的接口才可以访问，不能用from xxx import *导入；
    • 以双下划线开头的__foo代表类的私有成员；
    • 以双下划线开头和结尾的__foo__ 是特殊方法专用的表示，比如__init__()代表类的构造函数；
  • 要同一行写多条语句的话，语句间要用;分割。

7. 函数

7.1 参数传递

python 中类型属于对象，变量是没有类型的，比如a=[1,2,3]，[1,2,3] 是 List 类型，而变量 a 是没有类型，它仅是一个对象的引用（一个指针），可以指向 List 类型对象，也可以指向 String 类型对象。

可变类型和不可变类型

  • 可变类型 list, dict，变量赋值 la=[1,2,3,4] 后再赋值 la[2]=5 则是将 list la 的第三个元素值更改，本身la没有动，只是其内部的一部分值被修改了。
  • 不可变类型 string, tuple, number，变量赋值 a=5 后再赋值 a=10，这里实际是新生成一个 int 值对象 10，再让 a 指向它，而 5 被丢弃，不是改变a的值，相当于新生成了a。

python函数的参数传递：

  • 可变类型：类似 c++ 的引用传递，如 列表，字典。如 fun（la），则是将 la 真正的传过去，修改后fun外部的la也会受影响；
  • 不可变类型：类似 c++ 的值传递，如 整数、字符串、元组。如fun（a），传递的只是a的值，没有影响a对象本身。比如在 fun（a）内部修改 a 的值，只是修改另一个复制的对象，不会影响 a 本身。

7.2 参数

  • 必备参数：需要以正确的顺序传入函数，调用时的数量必须和声明时的一样；
  • 关键字参数：允许与声明的顺序不一致，因为解释器能用参数名匹配参数值；可以设置默认参数，默认参数如果没有传入，则被认为是默认值。
  • 不定长参数：如果需要一个函数能处理比声明时更多的参数时使用，声明时不会命名，*var_args_tuple，加了*的变量名会存放所有未命名的变量参数。

7.3 匿名函数lambda

  • lambda函数拥有自己的命名空间，不能访问自有参数列表之外或全局命名空间里的参数。
  • 不等同于C或C++的内联函数，后者的目的是调用小函数时不占用栈内存从而增加运行效率。

7.4 变量作用域

全局变量和局部变量
定义在函数内部的变量拥有局部作用域，定义在函数外的变量拥有全局作用域。

局部变量只能在其被声明的函数内部访问，而全局变量可以在整个程序范围内访问。调用函数时，所有在函数内声明的变量名称都将被加入到作用域中。

8. python模块

python 模块是一个.py的python 文件，里面定义函数、类或变量，以及可执行的代码，把相关的代码整理在一个模块里能让代码更清晰易懂，有助于有逻辑地组织python代码段。

8.1 import时的搜索路径

当解释器遇到import语句，会在当前的搜索路径搜索，找到则导入。搜索路径是一个解释器会先进行搜索的所有目录的列表，存储在 system 模块的 sys.path 变量中，变量里包含当前目录，PYTHONPATH和由安装过程决定的默认目录。所以搜索的顺序是：
1、当前目录
2、如果不在当前目录，则搜索在 shell 变量PYTHONPATH 下的每个目录。
3、如果都找不到，Python会查看默认路径。UNIX下，默认路径一般为/usr/local/lib/python/。

8.2 命名空间和作用域

命名空间是包含了变量名称（键）和对象（值）的字典，如果局部变量和全局变量重名，则局部变量会覆盖全局变量。

每个函数都有自己的命名空间。如果要给函数内的全局变量赋值，必须使用global varName，告诉python varnode 是一个全局变量，python就不会在局部命名空间中找这个变量了。

8.3 相关函数

8.3.1 dir() 函数

dir()返回的是一个排好序的字符串列表，存的是一个模块里定义过的名字，包括定义的所有模块、变量和函数。

8.3.2 globals() 和 locals()

globals() 和 locals()返回类型是字典，内容是当前调用位置的全局和局部命名空间中的名字。

8.3.3 reload()

一个模块只会被导入一次，不管执行了多少次import。用reload()可以重新导入之前导入过的模块。注意直接放模块名，而不是模块名的字符串：reload(test)
