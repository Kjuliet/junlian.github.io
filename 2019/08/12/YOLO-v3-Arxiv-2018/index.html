YOLO v3 (Arxiv, 2018)
发布于：2019-08-12 01:34:19 标签：/ 计算机视觉 / 目标检测 / 访问：

论文：YOLOv3: An Incremental Improvement
作者：Joseph Redmon, Ali Farhadi
YOLO v3的模型比之前的更复杂，在COCO测试集上mAP@0.5达到57.9%，与RetinaNet 的结构相近，但速度快4倍。YOLO v3一般比Fast R-CNN快100倍。

YOLO v3没有用Softmax对每个框分类
因为Open Images数据集的分类目标可能有重叠的类别标签，所以softmax不适用于多标签分类；Softmax 可以被独立的多个logistic分类器替代，且准确率不会下降；分类损失用的是binary cross-entropy loss。

多尺度预测

每种尺度预测 3 个box，anchor的设计方法还是用聚类，得到9个聚类中心，按照大小均分给3个尺度：

  • 尺度1：在基础网络之后先加几层卷积再输出box信息；
  • 尺度2：先从尺度1中的倒数第二层卷积上采样，再和最后一个16x16特征想加，再经过几个卷积后输出box，比尺度1变大两倍；
  • 尺度3：和尺度2类似，特征用的是32 x 32。

基础网络换为Darknet-53

Darknet-53 和 ResNet-101或ResNet-152准确率接近，但速度更快。

随着IOU的增大，YOLO v3性能会下降，说明它不能很好地与groud truth 切合。
