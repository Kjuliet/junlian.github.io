Visual Relationship Detection(VRD) 方法和应用
发布于：2019-08-05 20:56:04 标签：/ 计算机视觉 / 访问：

一、应用

Scene Graph Generation是从一张图像中得到scene graph的过程，scene graph可用于Image Retrieval, Image Generation, Visual Question Answering等任务，Visual Relationship Detection(VRD)是Scene Graph Generation中核心部分。

二、方法

先用Object detector (Fast R-CNN / Faster R=CNN, VGG / ResNet backbone) 检测objects, 抽取visual features，再预测object pairs间的关系（多分类任务）。
Deep Residual Learning for Image Recognition (CVPR 2016)
同时预测relationships和objects。
Scene Graph Generation by Iterative Message Passing (CVPR 2017) (baseline)
Visual translation embedding network for visual relation detection (CVPR 2017)
在某些预先设置的条件下建立关系图，通过结点间的信息传递来refine网络，目的是可以在 object 和 relationships之间交换信息
ViP-CNN: Visual Phrase Guided Convolutional Neural Network (CVPR 2017)
Scene Graph Generation from Objects, Phrases and Region Captions (ICCV 2017)
Detecting Visual Relationships with Deep Relational Networks (CVPR 2017)
用region features计算一个global embedding。
Neural Motifs: Scene Graph Parsing with Global Context (CVPR 2018) (baseline)
Deep Variation-structured Reinforcement Learning for Visual Relationship and Attribute Detection (CVPR 2017)
不用object detector，用一个embedding表示每个object，根据embedding间的相似性决定objects间的关系。
Pixels to Graphs by Associative Embedding (2017) (baseline)
利用额外信息：
（1）pre-trained word embeddings。
Visual Relationship Detection with Language Priors (ECCV 2016)
（2）利用训练集中的统计信息得到subject-object pairs的先验分布。
Neural Motifs: Scene Graph Parsing with Global Context (CVPR 2018)
Tensorize, Factorize and Regularize: Robust Visual Relationship Learning (CVPR 2018)
