1. 加法原理、乘法原理

分类计数原理：做一件事，有$n$ 类方法，第1类中有$m_1$种不同方法，第2类中有$m_2$种方法，…，第n类中有$m_n$种方法，那么完成这件事共有$N=m_1+m_2+…+m_n$种方法。
分步计数原理：做一件事，有$n$ 个步骤，第1步有$m_1$种不同方法，第2步有$m_2$种方法，…，第n步有$m_n$种方法，那么完成这件事共有$N=m_1\times m_2 \times … \times m_n$种方法。

2. 排列问题

从 n 个不同元素中取出 $m(m \le n)$ 个元素的所有不同排列的个数，叫做从 n 个不同元素中取出 m 个元素的排列数，记为 $A_n^m$。

按照分步乘法原理，取第一个有 n 种取法，第二个有 $(n-1)$ 种取法，…，第 m 个有 $(n-m+1)$种取法：

【性质1】 $A_n^m = nA_{n-1}^{m-1}$，先安排某个特定位置，再安排其他位置。
【性质2】 $A_n^m = mA_{n-1}^{m-1}+A_{n-1}^m$，某个特定元素的排列有$mA_{n-1}^{m-1}$，不含特定元素的排列有 $A_{n-1}^m$

\begin{equation}
\begin{split}
mA_{n-1}^{m-1}+A_{n-1}^m &= \frac{m(n-1)!}{(n-m)!} + \frac{(n-1)!}{(n-m-1)!} \\
&=\frac{m(n-1)!+(n-m)(n-1)!}{(n-m)!} \\
&= \frac{n!}{(n-m)!}
\end{split}
\end{equation}

3. 组合问题

从 n 个不同元素中取出 $m(m \le n)$ 个元素的所有不同组合的个数，叫做从 n 个不同元素中取出 m 个元素的组合数，记为 $C_n^m$。

排列问题$A_n^m$可看作两个步骤：第一步从n个元素中抽m个出来，先不排序，即组合问题 $C_n^m$，第二步把抽出的m个元素排序，即全排列$A_m^m$：根据分步乘法原理第一步第二步相乘得$A_n^m$

【性质1】$C_n^m=C_n^{n-m}$
【递推公式】$C_n^m = C_{n-1}^{m}+C_{n-1}^{m-1}$

对某一特定元素，最后取出的m个元素里要么有该元素，即$C_{n-1}^{m-1}$，表示已经取了一个，从剩下的n-1个元素中再取m-1个元素；要么没有，即$C_{n-1}^{m}$，表示一个都没取，还需要从n-1个元素中取m个。
