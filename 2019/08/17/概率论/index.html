1. 事件及运算

1.1 事件之间之间的关系

（文氏图）

  • 包含事件 $A\subset B$
  • 并（和）事件 $A \cup B = A + B$ A发生或B发生
  • 差事件： $A - B = A\bar B = A-AB$
  • 交（积）事件： $A\cap B$
  • 互斥事件：$AB=\oslash$ A发生B就不可能发生
  • 对立（逆）事件：$A\cup\bar A=S, A\bar A=\oslash$ 在整个集合空间，除了A就是A的逆
  • 独立事件：$P(AB)=P(A)P(B)$，可以认为A和B各自属于不同的集合，有三点需要注意：
    • 若A、B独立，$A, \bar B$、$\bar A, B$、$\bar A, \bar B$ 也相互独立；
    • 若A、B、C相互独立，则两两相互独立，$P(ABC)=P(A)P(B)P(C)$
    • 两两相互独立推不出A、B、C 相互独立。

互斥事件、对立事件和独立事件的区别：

  • 互斥事件和对立事件都是在一个集合中间，独立事件是分属不同的整体，在多个集合中；
  • 互斥事件和对立事件都是互不相交，不同在于对互斥事件，整个空间除了A、B还有其他事件，对立事件除了A就是B。

1.2 运算

德摩根律
$\overline{A\cup B}=\overline A \cap \overline B$
$\overline{A\cap B}=\overline A \cup \overline B$

加法公式
$P(A\cup B)=P(A)+P(B)-P(AB)$
$P(A\cup B\cup C) = P(A) + P(B) + P(C) - P(AB)-P(BC)-P(AC)+P(ABC)$

减法公式
$P(A-B)=P(A \overline B)=P(A)-P(AB)$

对立事件
$P(A)=1-P(\overline A)$

独立事件
$P(AB)=P(A)\cdot P(B)$

2.2 全概率公式

  • 设 $A$ 为发生的事件
  • 找出完备事件组 $B$
  • 写出 $P(B)$ 及 $P(A|B)$
  • 带入全概率公式：$P(A)=\sum\limits _{i=1}^n P(B_i)P(A|B_i)$

2.3 贝叶斯公式（逆概）

$P(B_i|A)=\frac{P(B_I)P(A|B_I)}{P(A)}$

【题1】甲、乙、丙三车间加工同一产品，加工量分别占总量 25%，35%，40%，次品率分别为 0.03，0.02，0.01，现从所有产品中抽取一个产品，求：
（1）该产品是次品的概率；
（2）若检查该产品是次品，求该产品是乙车间生产的概率。
解：
（1）该产品为次品记为事件A，甲厂生产记为$B_1$，乙场$B_2$，丙厂$B_3$
已知：$P(B_1)=1/4, P(B_2)=7/20, P(B_3)=2/5$
$P(A|B_1)=0.03, P(A|B_2)=0.02, P(A|B_3)=0.01$
则：$P(A)=P(B_1)P(A|B_1)+P(B_2)P(A|B_2)+P(B_3)P(A|B_3)=37/2000$
（2）$P(B_2|A) = P(B_2A)/P(A)=P(A|B_2)P(B_2)/P(A)$

【题2】盒中有4个红球，6个黑球，随机取出一球，观察颜色后放回，同时加入同色求2个，再从盒中第二次抽取一球，求：

（2）已知第二次取出的是黑球，求第一次取出的也是黑球的概率。
解：
（1）第二次取出黑球记为事件A，$B_1$为第一次取出红球，$B_2$为第二次取出黑球；
已知：$P(B_1)=4/10, P(B_2)=6/10，P(A|B_1)=6/12，P(A|B_2)=8/12$
则：$P(A)=P(A|B_1)P(B_1)+P(A|B_2)P(B_2)$
（2）$P(B_2|A)=P(B_2A)/P(A)=P(A|B_2)P(B_2)/P(A)$

3 一维随机变量

3.1 离散型随机变量

3.1.1 分布律、分布函数

分布函数 $F(x) = P\{X \le x\}$:

  • $0 \le F(x) \le 1$
  • $F(-\infty)=0，F(+\infty) = 1$
  • $F(x)$ 单调不减函数
  • 右连续 $\mathop{lim} \limits_{x\rightarrow x_0^+}(F(x))=F(x_0)$，所以分布函数写为左闭右开的形式

【题1】盒中有6个球，其中4个白球，2个黑球，从中任取2个球，求：
（1）抽到白球数 $X$ 的分布律；
（2）随机变量 $X$ 的分布函数
解：（1）$X$ 可取0，1，2

所以$X$的分布律为：
|X|0|1|2|
|—-|—-|—-|—-|
|P|1/15|8/15|6/15|
（2）随机变量$X$的分布函数：
$x \lt 0$ 时，$F(X)=0$
$0 \le x \lt 1$ 时，$F(X)=1/15$
$1 \le x \lt 2$ 时，$F(X)=9/15$
$2 \le x $ 时，$F(X)=1$
所以分布函数为：
\begin{equation}
F(x)=\left\{
\begin{array}{lr}
0 &x \lt 0 \\
1/15 &0 \le x \lt 1 \\
9/15 & 1 \le x \lt 2 \
1 & 2 \le x
\end{array}
\right.
\end{equation}

3.2 连续型随机变量

3.2.1 概率密度

连续型随机变量的概率密度$f(x)$对应离散型随机变量的分布律：

  • $\int_{-\infty}^{+\infty}f(x)dx = 1$
  • $F(x)=P\{X\le x\}=\int_{-\infty}^xf(x)dx$
  • $P\{a\le X\le b \} = F(b) - F(a)=\int_a^bf(x)dx$

【题1】设连续型随机变量$X$的概率密度函数为：
\begin{equation}
f(x)=\left\{
\begin{array}{lr}
a+x^2 & 0\le x \le 1\\
0 & 其他
\end{array}
\right.
\end{equation}
求：（1）常熟 $a$；（2）$P\{X \ge 0.5\}$；（3）分布函数$F(x)$
解：（1）
\begin{equation}
\begin{array}{ll}
\int_{-\infty}^{+\infty} f(x) dx&=& \int_{-infty}^0 0dx + \int_0^1(a+x^2) dx + \int_1^{+\infty}0dx \\
&=&\int_0^1(a+x^2) dx \\
&=&\Big(ax+\frac{1}{3}x^3\Big)\Big|_0^1 =1\\
&=>& a = \frac{2}{3}
\end{array}
\end{equation}

（2）
\begin{equation}
\begin{array}{ll}
P\{X \ge 0.5\} &=& \int_{0.5}^{+\infty}\Big(\frac{2}{3}+x^2\Big)dx \\
&=&\int_{0.5}^{1}\Big(\frac{2}{3}+x^2\Big)dx \\
&=& \Big(\frac{2}{3}x+\frac{1}{3}x^3 \Big) \Big| _{0.5}^1 \\
&=& \frac{5}{8}
\end{array}
\end{equation}

（3）$x < 0$ 时，$F(x)=\int_{-\infty}^x 0dx = 0$
$ 0\le x \lt 1$时，$F(x) = \int_{0}^x \big(\frac{2}{3}+x^2\big)dx=\Big(\frac{2}{3}x+\frac{1}{3}x^3 \Big) \Big| _{0.5}^x = \frac{2}{3}x+\frac{1}{3}x^3$
$ 1\le x $时，$F(x) = \int_{-\infty}^{0} \big(\frac{2}{3}+x^2\big)dx+\int_{0}^{1} \big(\frac{2}{3}+x^2\big)dx+\int_{1}^{x} \big(\frac{2}{3}+x^2\big)dx=\Big(\frac{2}{3}x+\frac{1}{3}x^3 \Big) \Big| _{0}^{1} = 1$
综上：
\begin{equation}
F(x)= \left\{
\begin{array}{ll}
0 & x<0 \\
\frac{2}{3}x+\frac{1}{3}x^3 & 0\le x \lt 1 \\
1 & 1\le x
\end{array}
\right.
\end{equation}

4.1.1 二项分布 $X \sim b(n, p)$

记为：$X \sim b(n, p)$ 或 $X \sim B(n, p)$
分布律：$P\{X=k\} = C_n^k p^k(1-p)^{n-k}$

【题1】一大楼有 5 台供水设备，设每台设备是否被使用相互独立，同一时刻每台被使用的概率为 0.1，问在某一时刻：
（1）恰有两台设备被使用的概率；
（2）至少有两台设备被使用的概率。
解：$X \sim b(5, 0.1)$ ，分布律是 $P\{X=k\}=C_5^k(0.1)^k \cdot (0.9)^{5-k}$
（1）$P\{X=2\} = C_5^2 (0.1)^2 (0.9)^3 = 0.0729$
（2）$P\{X \ge 2\} = 1 - P\{X \lt 2\} = 1-P\{X=0\}-P\{X=1\} = 0.08146$

【题2】设$X \sim B(2, p), Y \sim B(3, p)$, 若 $P\{X \ge 1\}=\frac{5}{9}$，则$P\{Y \ge 1\} = ？$

记为：$X \sim \pi (\lambda)$

4.2.3.1 标准正态分布

记为：$X \sim N(0, 1)$
概率密度：$\frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}$
特征：

  • 分布函数 $\Phi(x) = P\{X \le x\} = \int_{-\infty}^x f(x) dx$
    • $\Phi(0) = \frac{1}{2}$
    • $\Phi(-x) = 1 - \Phi(x)$
  • 若 $X \sim N(\mu, \sigma ^2)$
    • $P\{X<a\} = \Phi(\frac{a-\mu}{\sigma})$
    • $P\{a<X<b\} = \Phi(\frac{b-\mu}{\sigma}) - \Phi(\frac{a-\mu}{\sigma})$
