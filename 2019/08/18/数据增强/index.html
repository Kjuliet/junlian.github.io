3. 有监督增强方法

增强因子B：数据做离散增强后增长的倍数

3.1 单样本空间几何变换

只选择图像的一部分，或者做像素的空间重分布，没有改变图像本身的内容。

2.1.1 翻转

  1. iaa.Fliplr(0.5)：水平镜面翻转，参数为每张图片翻转概率
  2. iaa.Flipud()：上下翻转

2.1.2 裁剪

随机从图像中选择一部分裁剪出来，一般会调整成圆图像大小

  1. iaa.CropAndPad()：crop裁剪 或 pad填充：
  • px / percent：px和percent二选一，为负时表示crop, 为正表示pad, 用int或int list，或4元素的元组(top, right, bottom, left),原色可以是int/int tuple/int list；
  • pad_mode：可选constant,edge,linear_ramp, maximum, median, minimum, reflect, symmetric, wrap；
  • pad_cval：当pad_mode=’constant’时填充的值，float,int,tuple,list；
  • keep_size：经过crop和padding后，是否再缩放回原来的大小
  1. iaa.Pad()：同 iaa.CropAndPad() 接受positive value
  2. iaa.Crop()：同 iaa.CropAndPad() 接受negative value，如iaa.Crop(percent=([0.0, 0.05], [0.0, 0.1], [0.0, 0.1], [0.0, 0.1]))

2.1.3 缩放

放大或缩小图像尺寸，大多数会按照原始尺寸对放大后的图像进行裁剪，缩小时需要对图像边界之外的内容做假设。

  1. iaa.Scale()：把图像缩放到某一大小。

2.1.4 旋转

顺时针或逆时针旋转，注意最好旋转 90 / 180度，否则会出现尺度问题，B=2/4

2.1.5 平移

将图像沿x或y方向移动，需要对背景做假设，因为平移时有一部分图像是空的，因为图片中物体可能出现在任意位置，所以平移增强方法很有用。

2.1.6 仿射变换

同时做裁剪、旋转、转换等多重操作。

  1. iaa.Affine()：放射变换，包括 translation平移，Rotation 旋转，zoom 放缩，shear 错切，通过cval和mode指定放射变换生成的新的像素点的值，用offer 指定插值方法。
  • scale：图像缩放因子，1 不缩放，0.5 缩小为原图一半，x, y 可指定不同的缩放比例；
  • translate_percent：正负表示平移方向，大小表示平移比例，0不平移，0.5平移50%；
  • translate_px：按像素点平移；
  • rotate：旋转角度，0-360度，正负表示方向；
  • shear：错切的程度，0-360度，正负表示方向；

  • order：插值顺序，

    • 0：最近邻插值，快；
    • 1：双线性插值，快，默认；
    • 2：双二次插值，不推荐；
    • 3：双三次插值，比较慢；
    • 4：Bi-quartic，特别慢；
    • 5：Bi-quintic，特别慢；
    • ia.ALL：每次从所有插值方法中随机选。

  • mode：指定空白处的填充方式：

    • constant：常量填充；
    • edge：边缘填充；
    • symmetric：镜面对称填充；

  • cval：

    • int / float / tuple：当mode=constant时指定填充常量值；
    • ia.ALL：从 [0, 255] 之间随机选一个值填充。

2.1.7 四点透视变换

iaa.PiecewiseAffine()：随机设置网格点，移动网格点周围的像素使得局部扭曲；

2.1.8 分段仿射变换

在图像上放置一个规则的点网格，根据正态分布的样本数量移动这些点及周围的图像区域。

变化更多样，改变了像素内容

2.2.1 亮度

iaa.Multiply()：每个像素点乘一个值使得图片更亮或者更暗：iaa.Multiply((0.5, 1.5))

2.2.2 对比度

iaa.ContrasNormalization()：对比度：iaa.ContrastNormalization(alpha=1.0, per_channel=False)

2.2.3 灰度

iaa.Grayscale()：变灰度图：iaa.Grayscale(alpha=0,from_colorspace='RGB')

2.2.4 噪声

为了消除高频特征的影响，随机加入噪声数据消除这些特征，噪声类型有高斯、椒盐等

  1. iaa.Convolve()：对图像做卷积，matrix 参数是卷积矩阵：
  2. iaa.Sharpen()：锐化：iaa.Sharpen(alpha=0,lightness=1)
  3. iaa.Emboss()：浮雕效果：iaa.Emboss(alpha=0, strength=1)

2.2.5 模糊

  1. iaa.GaussianBlur(sigma=(0.0, 1.0))：高斯扰动，
  • sigma：高斯变换的标准差，float/float tuple，常见0不扰动，3强扰动
  1. iaa.AverageBlur(k=1)：取最近邻像素的均值扰动：
  • k：int / int tuple，窗口大小，当 k 为 int tuple 时，如果每个元素也是tuple，则分别表示height和width窗口大小不同。
  1. iaa.MedianBlur(k=1)：用最近邻中位数扰动：k 参数同上

2.2.3 颜色扰动

  1. iaa.ChangeColorspace()：改变颜色空间，iaa.ChangeColorspace(to_colorspace='HSV',from_colorspace='RGB', alpha=1.0)
  • Alpha：int, float, tuple，覆盖旧的色彩空间时，新颜色空间的Alpha值，
