1. 行列式性质及计算

行列式是一个函数，它可以将方阵（注意只有方阵才有行列式）映射到一个实值，他等于矩阵特征值的乘积，也就是说，他的大小可以衡量矩阵变换后空间扩大或者缩小的情况。比如：如果行列式为0那么说明空间至少沿着某一维完全收缩了，使其失去了所有体积，行列式为1则说明矩阵变换没有改变空间体积。

  • 按行展开：$D = a_{i1} A_{i1} + a_{i2} A_{i2} + … + a_{im} A_{im} (i=1,2,3,…, n)$
  • 按列展开：$D = a_{1j} A_{1j} + a_{2j} A_{2j} + … + a_{mj} A_{mj} (j=1,2,3,…, n)$

3. 矩阵

3.1 行列式和矩阵的区别

  • 行列式是一个数，矩阵是一个表；
  • 行列式是 n x n 阶，矩阵是 n x m 阶；
  • $\lambda |A|$ 是把行列式某行/列 乘以 $\lambda$，$\lambda A$ 是把矩阵里每个元素都乘以 $\lambda$；
  • 行列式加减是数的运算，矩阵加减只能是同型矩阵，对应元素的加减；
  • 矩阵如果是方阵时，有行列式值。

3.2 三则运算

  1. 矩阵加减：
  • 同型矩阵，同行同列；
  • 对应元素相加减。
  1. 矩阵的数乘 $\lambda A$
  • 每个元素均要乘以 $\lambda$
  1. 矩阵的乘法
  • 前行乘后列：$A_{m\times n} \cdot B_{n\times s} = C_{m \times s}$
  • $AB \ne BA $时：$(A \pm B)^2 \ne A^2 + 2AB + B^2 , A^2 - B^2 \ne (A+B) (A-B) $
  • $AB = BA $时：$(A \pm B)^2 = A^2 + 2AB + B^2 , A^2 - B^2 = (A+B) (A-B) $

3.3 转置矩阵 $A^T$

行变列，列变行；
性质：

  • $(AB)^T = B^T A^T$
  • $(A^T)^T = A$
  • $(kA^T)=kA^T$
  • $(A\pm B)^T = A^T \pm B^T$

3.4 伴随矩阵 $A^*$

$A_{ij}$ 代数余子式
性质：

  • $(AB)^ = B^A^*$
  • $|A^*| = |A|^{n-1}$
  • 当A 可逆时，$A^*=|A|A^{-1}$
  • $(kA)^ = k^{n-1}A^$

3.5 单位矩阵

  • 主对角线全为1，其他位置全0
  • 单位矩阵的行列式值为1， $|E| = 1$
  • 任何矩阵乘单位矩阵都等于它本身，$EA = AE = A$

3.6 逆矩阵 $A^{-1}$

若 $AB = BA = E$，则 $B$ 为 $A$ 的逆矩阵，记 $B=A^{-1}$，即：$AA^{-1} = E$

  • $A^{-1} = \frac{A^*}{|A|}$
  • A 可逆的充要条件：$|A| \ne 0$

性质：

  • $(AB)^{-1} = B^{-1}A^{-1}$
  • $(kA)^{-1}=\frac{1}{k}A^{-1}$
  • $(A^{-1})^{-1} = A$
  • $(A^T)^{-1} = (A^{-1})^T$

3.7 矩阵的行列式

  • $|A| = |A^T|$
  • $|kA| = k^n|A|$
  • $|AB| = |BA| = |A||B|$
  • $|A^{-1}| = \frac{1}{|A|}$

  • $|3A| = 3^3 |A| = 54$
  • $|A^{-1} = frac{1}{|A|} = frac{1}{2}$
  • $|A^*| = |A|^{3-1} = 4$
