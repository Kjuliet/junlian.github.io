集合

1 元素和集合

  • 研究的对象叫元素，小写字母表示；一些元素组成的总体叫集合，大写字母表示。

  • 集合的三个特点

    • 确定性：比如高个子人，班里漂亮女生（即使没女生也不能叫集合）都不叫集合
    • 互异性：$\{x|x^2-2x+1=0\} = \{x=1\}$
    • 无序性：{1, 2, 3} 和 {3, 2, 1} 是一个集合

  • 集合相等：元素和元素个数一样，但顺序不一定相等。

  • $a\in A$ 元素 a 属于集合 A；$a\notin A$ 元素 a 不属于集合 A；

  • 集合表示方法：

    • 列举法：${1, 3, 5, …}$ 无限集合也可以列举法表示；
    • 描述法：用集合所含元素的共同特征来表示集合，$\{x|x是拖拉机\}$；

  • 常用数集表示方法

    • 非负整数 / 自然数 N：$\{0, 1, 2, …\}$
    • 正整数 $N^+$ / $N^*$
    • 整数 Z $\{0, 1, -1, 2, -2, …\}$
    • 有理数 Q $\{x|x=\frac{p}{q}, p, q \in Z\}$，可以表示成两个整数的比；

  • 子集：如果集合 A 中任何一个元素都是集合 B 中的元素，称 A 是 B 的子集，或 A 包含于 B，$A \subseteq B$

    • $A = B$：当$A\subseteq B$ 且 $B \subseteq A$
    • $A\subsetneq B$：A是B的真子集，$A \subseteq B$ 但 $A \ne B$
    • 任何集合都是它自身的子集 $A\subseteq A$
    • $\{\emptyset\}$ 不是空集 $\emptyset$，空集是任何集合的子集；
    • 一个含有 n 个元素的集合，有 $2^n$ 个子集。

集合论
德国数学家康托尔提出，“在康托尔之前的数学不叫数学，那只是一种逻辑，而且是一种不完善的逻辑。”

康托尔提出“任何集合都是它自身的子集 $A\subseteq A$”观点后造成了第三次数学危机，因为假如有一个集合是这么定义的：“所有不包含自身的集合的集合”，如果有自身，那么和定义矛盾；如果不包含自身，那他和性质矛盾了。

这样的集合是存在的，罗素：有一个理发师，在门口立牌子，“我只给自己不刮胡子的人刮胡子”，请问他给不给自己刮胡子？

最后康托尔疯掉了。叔本华和黑格尔辩哲学，辨不过也疯了，养了条黑色的狗叫“黑格尔”每天溜。

（1）$A = \emptyset$，$a>b$
（2）$A \ne \emptyset$，$a \le b$ and $b \le d$ and $c \le a$

闰年：
能被4整除且不能被100整除，或者能被400整除

1000年中能被4整除的有2500个 - 能被100整除的100个 + 能被400整除的25个 = 2425

地球公转周期是365.2422，比365多1/4天，所以每4年多一个闰年，但如果每四年多一个闰年是不行的，因为每四年多一个闰年相当于是按365.25算，100年会多出1天来，1000年多100天呐。这样设计后，10000年中，每一年多了0.2425天，和0.2422相比，10000年差了3天，可以接受。

2. 函数

定义：A中任何一个数在B中都有唯一的数$f(x)$与之对应，称为$A\rightarrow B$的函数，记作$y=f(x), x\in A$

3. 基本初等函数

3.1 对数函数

定义：如果$a^x=N(a>0 且 a\ne 1)$，则$x$叫做以$a$为底$N$的对数，记作$x=log_aN$。

  • $N$ 叫真数，$N>0$；
  • $a$ 叫底数，$a>0 且 a\ne 1$。
  • $log_a1=0$
  • $log_aa=1$

常用对数以10为底，记作 $lgN = log_{10}N$
自然对数以e为底，记作 $lnN = log_eN$，$e=2.71828…$

伽利略：给我时间、空间和对数，我可以创造一个世界。

对数可以把一个很大的数换成很小的数， $lg1000 = 3$，所以在工程上很有用。
