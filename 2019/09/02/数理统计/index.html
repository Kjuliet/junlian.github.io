3. $E(X), D(X)$的性质

3.1 期望 $E(X)$ 的性质：

  • $E(C) = C$
  • $E(CX) = CE(X)$
  • $E(X \pm Y) = E(X) \pm E(Y)$
  • $E(XY) = E(X)E(Y) $，当 X 和 Y 相互独立时

3.2 方差 $D(X)$ 的性质：

  • $D(C) = 0$
  • $D(CX) = C^2D(X)$
  • $D(X \pm Y) = D(X) + D(Y)$，当 X 和 Y 相互独立时

4.3 均匀分布 $U[a, b]$

  • $E(X) = \frac{a+b}{2}$
  • $D(X)=\frac{(b-a)^2}{12}$
  • $P(c\le X \le d) = \frac{d - c}{b-a}$

4.4 指数分布 $E(\lambda)$

  • $E(X) = \frac{1}{\lambda}$
  • $D(X)=\frac{1}{\lambda ^2}$
  • $P(c\le X \le d) = \frac{1}{e^{c\lambda}} - \frac{1}{e^{d \lambda}}$

4.5 正态分布 $N(\mu, \sigma ^2)$

  • $E(X) = \mu$
  • $D(X)=\sigma ^2$
  • $P(c\le X \le d) = \Phi(\frac{d-u}{\sigma}) - \Phi(\frac{c-u}{\sigma}) $
