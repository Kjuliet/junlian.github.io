    Activation functions

    Non-linear activation function
    If we use linear activation function

    当输出y是房子的价格时，输出层可以用线性activation functions，但中间层还是不能用。

    Sigmoid function


    $a=\frac{1}{1+e^{-z}}$

    • Going within zero and one.
    • If output is 0,1 value, if we’re using binary classification, then the sigmoid activation function is a natural choice for the output layer.
      Derivatives

    Tangent function/hyperbolic tangent function


    $a=tanh(z)=\frac{e^z-e^{-z}}{e^z+e^{-z}}$

    • Going between plus 1 and minus 1.
    • Actually mathematically a shifted version of the sigmoid function
    • For hidden units, if we let the activation function be equal to tanh(z), this almost always works better than the sigmoid function. The mean of your data is closer to 0 rather than 0.5.

    Derivatives

    ReLU(rectify linear unit)


    $a=max(0,z)$

    • ReLU is increasingly the default choice of activation function.
    • In practice, enough of hidden units will have z greater than 0, so learning can still be quite fast for most training examples.
    • So if you’re not sure what to use for hidden layer, just use the ReLU activation function.

    Leaky ReLU

    $a=max(0.01z,z)$

    • It takes a slight slope like.
    • Usually working better than the ReLU.

    Random Initialization

    b 可以初始化为0，w不能，否则隐藏层会平行symmetry problem

    Initialization with 0:

    Initialize it to a very small random values.

    Use small values

    • We are using a tanh or sigmoid activation function, or the other sigmoid, even just at the output layer.
    • If the weights are too large, large w make z, or at least some values of z very large or very small.
    • The slope is very small, meaning that gradient descent will be very slow.
