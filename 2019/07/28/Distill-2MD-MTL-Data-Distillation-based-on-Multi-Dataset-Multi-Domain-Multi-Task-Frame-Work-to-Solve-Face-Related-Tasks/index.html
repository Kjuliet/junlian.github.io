Distill-2MD-MTL: Data Distillation based on Multi-Dataset Multi-Domain Multi-Task Frame Work to Solve Face Related Tasks
发布于：2019-07-28 11:40:37 标签：/ 计算机视觉 / Face / 访问：

文章作者：Sepidehsadat Hosseini，Mohammad Amin Shabani，Nam Ik Cho
文章地址：https://arxiv.org/pdf/1907.03402.pdf

对和人脸相关的任务，论文提出了基于Multi-Task Learning 和 data distilation的半监督学习方法，通过挖掘不同但相互有关联的任务中带有不同标签的不同数据集中的信息，使得其中只标注了一少部分数据的某一任务得到更好的学习。也就是说，如果目标任务的数据集只有少量标注，通过利用与该任务相关的其他域的数据集可以提高准确率。

网络结构分为两部分，一个是用于处理weakly labeled数据集、且同时执行多个任务的新的Multi-Task Learning，另一个是基于Multi-Task Learning的数据蒸馏，能够提高网络在不同域上的泛化能力。

论文在共享层后加了一个discriminator head,为了让来自相同数据集的特征相近，构造了triplets $ T_i = (x^a, y^p, y^n)$ ，$x^a$ 是anchor, $y^p$ 是和 $x^a$ 同一数据集的图片，$y^n$ 是与 $x^a$ 不同数据集的图片，最后的loss由triplet loss 和 各任务的总loss两部分组成

实验结果发现多任务的结果比单个任务的结果好
